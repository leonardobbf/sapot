# sapot
O Protocolo para Sensoriamento e Atuação de Coisas (SAPoT) é um leve e
enxuto protocolo de fonte aberta desenvolvido para traduzir, de maneira simples e
direta, mensagens em uma rede IoT no modelo Central/Sentinela. Portanto, o
SAPoT tem por objetivo integrar as partes de uma rede multi protocolada.
